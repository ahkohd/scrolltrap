# Scrolltrap

Prevent page from scrolling after having reached the end of a scrollable element


### Prerequisites

None


### Installing

**Npm**

```
npm install --save scrolltrap
```

**Bower**

```
bower install scrolltrap
```

Or grab the file directly

```
<script src="app.js"></script>
<script src="app.js"></script>
```
